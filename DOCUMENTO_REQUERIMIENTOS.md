# Documento de Toma de Requerimientos

## Chatbot Vocacional con Inteligencia Artificial usando Dialogflow y Red Neuronal en Python

Este repositorio fue adaptado para representar un prototipo de orientación vocacional con Dialogflow, Flask y un modelo neuronal en Python/TensorFlow.

## Componentes implementados en el export

- Intents principales:
  - `Default Welcome Intent`
  - `Respuesta_Test`
  - `Finalizar_Test`
  - `Respuesta_Invalida`
  - `Cancelar_Test`
- Entities:
  - `@respuesta_valor`
  - `@confirmacion`
  - `@emociones_usuario`
- Flujo conversacional por contextos `pregunta_1` ... `pregunta_25`.
- Integración por webhook activada en intents de captura/finalización.
